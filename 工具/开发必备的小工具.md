# SwitchyOmega

SwitchyOmega是一款chrome代理扩展插件，可以轻松管理和切换多个代理设置。

优点：不同于其他代理软件只能选择走代理，该插件在遇到个别网站需要全局代理时，可自动切换。

![](https://blog.airaccoon.cn/img/bed/20190520/1558316681803.png)



# whistle

结合SwitchyOmega成为一款前后端开发神器。

whistle是一款抓包工具。

## 安装

```shell
 npm install  -g  whistle
 
 w2 -h
 
 # 启动，不设置端口默认使用8899
 w2 start -p 8899

```
