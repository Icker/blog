# whistle

[whistle文档](https://github.com/avwo/whistle/blob/master/README-zh_CN.md)

结合SwitchyOmega成为一款前后端开发神器。

whistle是一款抓包工具。

## 安装

```shell
 npm install  -g  whistle
 
 w2 -h
 
 # 启动，不设置端口默认使用8899
 w2 start -p 8899

```
